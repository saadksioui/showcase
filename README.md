# Saad Ksioui's Portfolio

> Designing thoughtful digital products with clarity, craft, and calm execution.

A modern, animated portfolio website showcasing my work as a Software Engineer. Built with Next.js, React, and Tailwind CSS to demonstrate clean code, thoughtful design, and impactful digital solutions.

## Features

- 🎨 **Beautiful UI** - Built with shadcn/ui components and Tailwind CSS
- ✨ **Smooth Animations** - Powered by Motion library for elegant transitions
- 🌓 **Dark Mode Support** - Built-in theme switching with next-themes
- 📱 **Responsive Design** - Mobile-friendly layout that works on all devices
- 🔗 **GitHub Integration** - Display your contribution graph and stats
- ⚡ **Fast Performance** - Next.js 16 with TypeScript for optimal speed
- 🎯 **Sections Included** - Hero, About, Tools, Projects, Experience, and more

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) 16.2.7
- **UI Library**: React 19.2.4 with [shadcn/ui](https://ui.shadcn.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) 4
- **Animations**: [Motion](https://motion.dev/)
- **Package Manager**: [Bun](https://bun.sh/)
- **Icons**: React Icons, Iconify, and Lucide React
- **Type Safety**: TypeScript 5

## Getting Started

### Prerequisites

- Bun (or Node.js)
- Git

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd showcase

# Install dependencies
bun install
```

### Development

```bash
# Start the development server
bun run dev
```

The application will be available at `http://localhost:3000`

### Build for Production

```bash
# Build the application
bun run build

# Start production server
bun start
```
